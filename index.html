Una Guia para que encuentres tu camino como desarrollador Front FrontEnd

En el desarrollo Front End te econtrarás con una gran variedad de caminos, lenguajes, frameworks y librerías , en este web site te resumiré las tendencias del momento y sus características para que así escogas el camino que mas te llame la atención

HTML

El lenguaje html (hypertext markup language) se utiliza para el desarrollo y creación de páginas web. Se compone de una serie de etiquetas que el navegador interpreta. Entre las etiquetas que se incluyen dentro del lenguaje HTML se encuentran: hipervínculos, etiquetas para imágenes, saltos de página, entre otras.

arrow_drop_down_circle arrow_drop_down_circle

CSS

CSS (en inglés Cascading Style Sheets) es lo que se denomina lenguaje de hojas de estilo en cascada y se usa para estilizar elementos escritos en un lenguaje de marcado como HTML. CSS separa el contenido de la representación visual del sitio.

arrow_drop_down_circle arrow_drop_down_circle

JAVASCRIPT

JavaScript es el único lenguaje de programación que funciona en los navegadores de forma nativa (lenguaje interpretado sin necesidad de compilación). Por tanto se utiliza como complemento de HTML y CSS para crear páginas webs.
